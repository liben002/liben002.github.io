Hackathon Projects

Wikiwhere - Apr 2019

Have you ever heard about the Six Degrees of Separation? It says that all people are at most only six social connections away from each other. Based on this idea, Wikiwhere finds the shortest path from one Wikipedia article to another using only article links found in each intermediate page. Implemented with a double-ended Breadth-First Search algorithm in the backend, and D3.js for the user interface, Wikiwhere shows that any two pages are only six clicks away from each other. This project won 1st place at the 2019 Codestellation hackathon. Link to the GitHub repo.

hackm.app - Nov 2018

hackm.app is a service that displays MLH-registered hackathons on a map. I was responsible for the web scraper, which retrieved hackathon info from the MLH site using Python and the BeautifulSoup library. This project led me to my first Software Engineering internship. Link to the GitHub repo.
