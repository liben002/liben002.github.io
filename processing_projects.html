Processing Projects

Google Dinosaur Game

The final project for APCS! Wow, this takes me back. At the time, I, a junior in high school,had less than a year of programming experience. Yet I was able to completely recreate the Google dinosaur game, which was a huge accomplishment for me at the time. It's not very well optimized, at all, and there are many bugs that sometimes causes the game to crash. Nevertheless, I had a great time making it, and it's a fun thing to show others. Link to GitHub repo. I am trying to get this to work on OpenProcessing so that it can be embedded onto this page.

Random Fractal Tree

Finally, something that won something. This random fractal generator was my submission to the Hour of Code 2017 competition. I got the idea from combining two labs Dr. Nguyen had us do towards the end of the year, which is probably why the project is a little basic. However, it works well and took a lot of research to complete, so I am proud of it. It's also easily configurable to change the base shape and pattern, giving it flexability. This won second place in the 3D Modeling Category, so maybe I should make some prints. Link to GitHub repo.
